### Installation Shark
* `Commands` for termux
```
$ termux-setup-storage

$ pkg update && pkg upgrade && pkg install git -y

$ git clone https://github.com/Bhaviktutorials/shark

$ ls

$ cd shark

$ ls

$ chmod +x *

$ ./requirements.sh

$ shark
```
